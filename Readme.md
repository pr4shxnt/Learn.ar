## ‚öôÔ∏è Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/stha-sanket/TechSprint_visionary.git
   cd TechSprint_visionary
   ```

2. **Server Setup:**
   ```bash
   cd server
   npm install
   # Create a .env file and add your MongoDB_URI, GEMINI_API_KEY, and PORT
   npm start
   ```

3. **Client Setup:**
   ```bash
   cd ../client
   npm install
   npm run dev
   ```
